## AIが事業を選ぶ基準

棚卸しが終わると、AIは複数の事業案を比較して最適なものを選ぶ。このとき判断基準を明示しておくと、選定の精度が上がる。

実験では以下の8軸で比較させた。

```
比較軸：
1. 利益率
2. 初回販売までのスピード
3. 初期コスト
4. 競合の少なさ
5. 市場規模
6. 既存資産の活用度
7. AI実行可能率（人間の作業が少ないか）
8. 継続収益の可能性
```

## 実験での5案比較

| 案 | 内容 | 結果 |
|---|---|---|
| A | ChatGPT×営業プロンプト集（500円） | ✅ 採択 |
| B | AI実験ログ記事（無料・集客用） | ✅ 採択 |
| C | マーケティングテンプレート集 | 保留 |
| D | 副業・フリーランス向けAI記事 | ❌ 却下 |
| E | コンサル集客LP | ❌ 却下 |

**Aを採択した理由**：完成に近い素材がすでにあり、即日販売可能。利益率が高く（価格500円、追加コストほぼゼロ）、AI実行可能率が高い。

**Dを却下した理由**：競合記事が多すぎて検索流入が見込めない。差別化に時間がかかりすぎる。

**Eを却下した理由**：初回収益まで時間がかかりすぎる。問い合わせ対応に人間の工数が必要になる。

## 事業選定のプロンプト

```
以下の資産リストと条件を踏まえて、事業案を5つ比較してください。

【資産】
[棚卸し結果を貼り付け]

【条件】
- 2週間以内に初回収益を出す
- 人間の作業は週5分以内に収める
- 追加費用は発生させない

各案を以下の8軸で点数評価し（各10点満点）、総合点と採択/却下の判断を出してください：
利益率・スピード・コスト・競合・市場・資産活用・AI実行可能率・継続性
```

## 選定後にやること

採択案が決まったら、AIに次を指示する。

```
案Aを採択します。
制作に入る前に、以下を確認してください：
1. 完成までの工程リスト
2. 人間が承認・確認が必要なステップ
3. 品質基準（何をもって完成とするか）

確認できたら制作を開始してください。
```

工程と品質基準を先に合意しておくことで、完成後の「思ってたのと違う」を防げる。
