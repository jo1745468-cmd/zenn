## AIに制作させる前に決めること

事業案が決まったら、すぐ「作って」と言いたくなる。でもその前に3つを決めておく。これを省略すると、出てきたものが使い物にならなくて作り直しになる。

**決めておくこと**
1. ターゲット（誰のための何か）
2. 品質基準（何をもって完成とするか）
3. 出力形式（どんな形で欲しいか）

## ターゲット設定のプロンプト

```
制作を始める前に、ターゲット設定を確認してください。

このコンテンツのターゲット：
- 誰：[例：20〜40代の営業職・BtoB営業3〜10年目]
- 状況：[例：ChatGPTは知っているが実務での使い方がわからない]
- ゴール：[例：明日の業務ですぐ使えるプロンプトを手に入れたい]
- 価格感度：[例：500円なら即決できる層]

このターゲットに最も刺さる構成・文体・見出しで制作してください。
```

## 品質基準の設定

品質基準をあらかじめ渡しておくと、AIが自分で基準に照らして修正できるようになる。

```
以下の品質基準を満たしてから完成とみなしてください：

【必須条件】
□ ターゲットの言葉で書かれている（専門用語は最小限）
□ コピペしてすぐ使える形式になっている
□ 各プロンプトに「どんな時に使う」が明示されている
□ AI生成特有の均一な文体になっていない
□ 重複・水増しがない

【禁止事項】
□ 根拠のない数字・統計の使用
□ 実体験として書けない内容の一人称使用
□ 他社・他サービスの誹謗中傷
```

## 制作指示のテンプレート

```
以下の仕様でコンテンツを制作してください。

【形式】
- 文字数：[例：8,000字前後]
- 構成：[例：はじめに → カテゴリ別プロンプト → おわりに]
- 価格：[例：500円の有料記事]

【制作後にやること】
1. 上記の品質基準でセルフチェックを実施
2. 問題があれば修正して再チェック
3. 全項目クリアしたら「完成」と報告

完成の報告と同時に、品質チェック結果も提出してください。
```
