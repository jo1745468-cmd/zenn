## プロンプト5｜提案書の構成案を作る

**どんな時に使う？**
商談後に提案書の骨格をすぐ作りたいとき。0から考え始めると時間のかかる「何を入れるか」の部分を一気に出力させます。

**プロンプト全文**

```
あなたはB2B営業のプロです。以下の情報をもとに提案書の構成案を作成してください。

■提案先：【業種・規模（例：中小製造業・従業員50名程度）】
■提案サービス：【商品名・価格帯（例：クラウド型在庫管理システム・月額3万円〜）】
■課題：【課題内容（例：在庫のダブり発注が月に数回発生している）】
■ページ数：【枚数（例：8枚）】

各セクションのタイトルと、そのページで伝えるべきポイントを箇条書きで出力してください。
```

**コツ**
課題欄が曖昧だと出力が凡庸になります。「毎月〇件発生している」「〇〇人が手作業で対応している」など、具体的な数字や事実を入れると質が上がります。

## プロンプト6｜商談メモから要約を作る

**どんな時に使う？**
商談後の生メモ（走り書き・箇条書き）から、報告や提案書に使える整理された要約を作るとき。

**プロンプト全文**

```
以下の商談メモを要約してください。

■商談メモ：【箇条書きや走り書きのまま貼り付けてください】

以下の形式で出力してください：
- 相手の現状・課題（2〜3点）
- 相手のニーズ・期待（2〜3点）
- 次回アクション（担当者と期日つき）
- 懸念点・確認事項（あれば）
```

**コツ**
メモが雑でも構いません。「前回も同じこと言ってた」「なんか担当者乗り気じゃなかった」のような主観的メモも入れてOKです。ChatGPTが整理してくれます。

## プロンプト7｜競合比較表を作る

**どんな時に使う？**
「他社とどう違うのか」を資料やメールで伝えたいとき。比較表の叩き台を素早く作ります。

**プロンプト全文**

```
以下の情報をもとに、自社サービスと競合サービスの比較表を作成してください。

■自社サービス名：【名前と主な特徴（箇条書き）】
■比較対象：【競合A：名前と特徴 / 競合B：名前と特徴】
■比較したい項目：【例：価格・導入期間・サポート・機能・実績など】

表形式（マークダウン）で出力してください。
自社が有利な点は強調し、不利な点は正直に記載してください。
```

**コツ**
「不利な点も正直に」と入れることで、資料として信頼性の高い比較表になります。都合のいいことだけ書いた比較表は顧客に見透かされます。

## プロンプト8｜競合情報を整理する

**どんな時に使う？**
競合他社について調べた情報（ホームページ・リリース・SNS等）を、商談前の準備資料としてまとめるとき。

**プロンプト全文**

```
以下の情報をもとに、競合会社の特徴と自社との比較ポイントをまとめてください。

■競合会社名：【会社名】
■収集した情報：【ホームページ・ニュース・口コミなど、調べたことを貼り付け】
■自社の強み：【主な強みを箇条書きで】

まとめる形式：
- 競合の強み（2〜3点）
- 競合の弱みまたは課題（2〜3点）
- 商談で活用できる差別化ポイント（2〜3点）
```

## プロンプト9｜数字・データを言語化する

**どんな時に使う？**
「導入後に〇〇が改善した」という数字やデータを、提案書やメールで使いやすい「ストーリー」に変えたいとき。

**プロンプト全文**

```
以下のデータを、顧客向けの提案に使える文章に変換してください。

■データ・数字：【例：導入後3ヶ月で処理時間が1件あたり45分→15分に短縮】
■伝えたいメッセージ：【例：効率化の具体的なイメージを持ってもらいたい】
■使用場面：【提案書の1スライド / 営業メールの1段落 / 口頭説明用など】

感情に訴えつつ、数字の根拠を失わない文章で書いてください。100〜150字程度。
```
