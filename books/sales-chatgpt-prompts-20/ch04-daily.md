## プロンプト13｜週次報告書をメモから作る

**どんな時に使う？**
毎週金曜の報告書提出。手元のメモから素早くまとめたいとき。

**プロンプト全文**

```
以下のメモをもとに、営業の週次報告書を作成してください。

■今週の訪問先と内容：【会社名・話した内容を箇条書きで】
■今週の数字：【訪問件数、商談件数、受注件数、売上など】
■来週の予定：【主なアポや対応予定を箇条書きで】

箇条書きを中心に、300文字程度でまとめてください。
上司への報告として読みやすい構成にしてください。
```

**コツ**
文字数指定を変えると分量が変わります。「500字」にすると詳細版、「200字」にすると要点のみの版になります。上司の好みに合わせて調整してください。

## プロンプト14｜議事録を箇条書きから作る

**どんな時に使う？**
商談・社内会議後の議事録をメモから作るとき。「誰が何を決めたか」が明確になる形式で出力させます。

**プロンプト全文**

```
以下の会議メモをもとに議事録を作成してください。

■会議名：【例：〇〇社との第2回商談】
■日時・参加者：【日時と参加者名（社名・役職）】
■会議メモ：【発言・決定・宿題などを箇条書きで】

以下の形式で出力してください：
- 決定事項（番号付き）
- 次回アクション（担当者・期日つき）
- 確認・検討が必要な事項（あれば）
- 次回会議の予定（わかれば）
```

**コツ**
「誰が」「いつまでに」を必ず入れることで、後から見ても使える議事録になります。メモにアクション担当者が書いてなければ、ChatGPTは「要確認」と出してくれます。

## プロンプト15｜週間スケジュールを整理する

**どんな時に使う？**
月曜朝にバラバラな予定・タスクを見て「今週何から手をつければいいか」を整理したいとき。

**プロンプト全文**

```
以下の情報をもとに、今週の優先順位付きスケジュールを作成してください。

■今週のアポ・予定：【日時と内容を箇条書きで】
■今週中に対応が必要なタスク：【箇条書き】
■締め切りのあるもの：【内容と期日】
■後回しにしていいもの：【あれば】

月〜金の日別に整理し、優先度（高・中・低）をつけてください。
```
