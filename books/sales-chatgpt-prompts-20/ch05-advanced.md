## プロンプト16｜ロールプレイで商談練習をする

**どんな時に使う？**
新規開拓の初回商談前、あるいは難しい顧客との商談に備えて、一人で練習したいとき。ChatGPTに顧客役をやらせます。

**プロンプト全文**

```
以下の設定で商談ロールプレイをしてください。

■あなたの役：【顧客の役（名前・役職・業種・性格）】
■私の役：営業担当
■商談の目的：【例：初回訪問でニーズをヒアリングする】
■顧客の設定：【例：ChatGPT導入に興味はあるが、情報システム部門の反対がありそう。慎重な判断をする人物】

私が話しかけるので、顧客として自然に反応してください。
商談が終わったら、「フィードバック」として私のトークへの評価を3点あげてください。
```

**コツ**
「慎重な人物」「懐疑的な人物」などのキャラクター設定を変えることで、さまざまなタイプの顧客への対策が練れます。1回の練習は10分程度で十分です。

## プロンプト17｜断られた後の切り返しトークを作る

**どんな時に使う？**
商談中に「今は必要ない」「高い」「他社で検討中」と言われたとき、圧力をかけずに会話を続けるための切り返し文を事前に準備しておくとき。

**プロンプト全文**

```
以下の状況での切り返しトークを3パターン作成してください。

■断りの内容：【例：「今期は予算がなくて…」】
■自社サービス：【名前と主な特徴】
■方針：【強引に押さず、次の打ち合わせへつなぎたい】

各パターンに「このトークが適しているシチュエーション」を一行で添えてください。
```

## プロンプト18｜提案書の「刺さる一文」を作る

**どんな時に使う？**
提案書の冒頭や表紙に入れる「この提案書の核心を一言で伝えるキャッチコピー」を作るとき。

**プロンプト全文**

```
以下の提案内容をもとに、提案書の冒頭や表紙に使える「刺さる一文」を5パターン作成してください。

■提案先の課題：【例：毎月の在庫ダブり発注で年間100万円以上のロスが発生している】
■提案内容の核心：【例：リアルタイム在庫共有で発注ミスをゼロにする】
■トーン：【数字で語る / 共感ファースト / 問いかけ形式など、好みがあれば】

各パターンに「このコピーが刺さる読者像」を一行添えてください。
```

## プロンプト19｜自社サービスのFAQ・想定反論を作る

**どんな時に使う？**
新しいサービスを売り始めるとき、顧客からよく聞かれる質問や反論を事前に洗い出して準備しておくとき。

**プロンプト全文**

```
以下のサービスについて、顧客から想定されるFAQと反論を作成してください。

■サービス名：【名前】
■サービスの概要：【何ができるか・誰向けか・価格帯】
■主なターゲット：【業種・規模・役職など】

以下を出力してください：
1. よくある質問（FAQ）5問と回答
2. 想定される反論・懸念点5つと、それぞれへの応答例
```

**コツ**
「厳しめの反論を作ってください」と追加すると、「それって本当に使えるの？」「同じようなツール、前に使ったけど続かなかった」のような、実際の商談に出てくる手強い反論が出てきます。

## プロンプト20｜月次振り返りと次月の方針を立てる

**どんな時に使う？**
月末の振り返りと翌月の目標設定を、感覚でなくデータから考えたいとき。

**プロンプト全文**

```
以下の営業データをもとに、今月の振り返りと来月の方針を作成してください。

■今月の数字：【訪問件数、商談件数、受注件数、売上額、目標比】
■うまくいった案件：【案件名と理由（簡単に）】
■うまくいかなかった案件：【案件名と理由（簡単に）】
■来月の数字目標：【訪問件数、受注件数、売上額など】

以下の形式で出力してください：
- 今月の総評（2〜3行）
- 来月に活かせる気づき（3点）
- 来月の優先アクション（3〜5点）
```
