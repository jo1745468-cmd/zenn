## プロンプト10｜業界・企業の基礎調査

**どんな時に使う？**
初めて会う企業や業界について商談前に手早く把握したいとき。知識がない状態でも「何を聞けばいいか」の仮説を立てる準備ができます。

**プロンプト全文**

```
以下の業界・企業について基礎情報を教えてください。

■業界：【例：中小製造業（自動車部品）】
■企業規模：【例：従業員50〜200名程度の中小企業】
■目的：【例：初回商談前の業界理解。課題を仮説立てするため】

以下の形式でまとめてください：
- 業界の主要課題（3点）
- この規模の会社が抱えやすい経営上の悩み（2〜3点）
- 商談で聞くべき確認事項（3〜5点）

※情報が不確かな場合は「一般的に」「傾向として」と断ってください。
```

**コツ**
ChatGPTの学習データに基づく回答なので、最新の業界動向は別途確認が必要です。「仮説を立てるための叩き台」として使う意識が重要です。

## プロンプト11｜ニュース・プレスリリースの要約

**どんな時に使う？**
訪問前に取引先や競合のプレスリリース・ニュースを読んでおきたいが、時間がないとき。

**プロンプト全文**

```
以下のニュース/プレスリリースを要約してください。

■テキスト：【記事本文をそのまま貼り付け】

以下の形式で出力してください：
- 要点（3行以内）
- 自社営業への影響・活用できるポイント（あれば）
- 商談で使えるトーク（あれば1〜2文）
```

**コツ**
「商談で使えるトークを出して」という指定を入れると、ただの要約ではなく使える情報になります。

## プロンプト12｜競合他社のリサーチ

**どんな時に使う？**
顧客から「〇〇社も検討しています」と言われたとき、相手の情報をすぐ把握して差別化ポイントを整理したいとき。

**プロンプト全文**

```
【競合会社名：○○株式会社】について、以下の観点から情報を教えてください。

1. 主力製品・サービスとその特徴
2. 強みとして知られている点
3. 弱みや顧客からの不満として言及されやすい点
4. 価格帯・ターゲット層

※情報が不確かな場合や2023年以降の情報は不正確な可能性があることを明示してください。
回答後、「この情報を踏まえて自社との差別化ポイント」を3点挙げてください（自社の強みは別途入力します）。
```
