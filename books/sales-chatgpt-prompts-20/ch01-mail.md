## プロンプト1｜商談後のお礼メール

**どんな時に使う？**
商談直後、移動中のスマホや帰社後に素早くお礼メールを送りたいとき。

**プロンプト全文**

```
以下の情報をもとに、商談後のお礼メールを作成してください。

■相手の会社名：【会社名】
■担当者名：【担当者名】
■商談日：【日付】
■商談の内容：【話した内容（箇条書きでOK）】
■トーン：【丁寧・ビジネスライクで。親しみやすさは控えめに】

件名と本文をセットで出力してください。
```

**出力例**

```
件名：本日のご面談のお礼（株式会社〇〇 △△より）

〇〇株式会社
□□様

本日はお忙しい中、お時間をいただきありがとうございました。

ご説明いただいた現状の課題について、改めて社内で整理のうえ、
来週中に具体的なご提案をお送りできればと考えております。

ご不明な点がございましたら、いつでもお気軽にご連絡ください。
引き続きよろしくお願いいたします。

株式会社〇〇
営業部　△△
TEL：xxx-xxxx-xxxx
```

**コツ**
「トーン」の指定を変えるだけで、相手の業種や関係性に合わせられます。「フォーマル」「少し親しみやすく」などを使い分けると直しが減ります。

## プロンプト2｜未返信先へのフォローアップメール

**どんな時に使う？**
提案書を送って1週間以上返事がない先への、催促にならない自然なフォローメールを作りたいとき。

**プロンプト全文**

```
以下の状況でフォローアップメールを作成してください。

■送付内容：【以前送ったもの（例：提案書、見積書）】
■送付日：【日付】
■相手の状況：【わかっている範囲で（例：検討中の様子、担当者が変わった可能性）】
■目的：【返信を促す / 次の打ち合わせを提案する / 期限確認など】

催促にならない、自然な言い回しで書いてください。件名と本文をセットで。
```

**コツ**
「催促にならない」という指定が重要です。省略すると圧力的な文章が出てきやすいため、必ず入れてください。

## プロンプト3｜お断りへの切り返し返信

**どんな時に使う？**
「今期は予算がなく…」「他社で進める方向で…」といった断りメールへの返信を作るとき。感情的にならず、次の接点を残す文章を作ります。

**プロンプト全文**

```
以下の状況でお断りへの返信メールを作成してください。

■断りの内容：【相手のメールの要旨（例：今期は予算確保が難しい旨）】
■自社のスタンス：【今後もアプローチを続けたい / いったん距離を置く】
■トーン：【感謝・共感を前面に、圧力をかけない文体で】

短め（200字以内）で書いてください。件名と本文をセットで。
```

**コツ**
「短め・感謝を前面に」を入れると、追いかける印象が出ません。断られたときほど短い返信が好印象です。

## プロンプト4｜社内関係者への報告・連絡メール

**どんな時に使う？**
商談結果や案件の進捗を上司・同僚・他部門に共有するとき。要点を漏れなく、読みやすくまとめたいとき。

**プロンプト全文**

```
以下の情報をもとに、社内向け報告メールを作成してください。

■報告相手：【上司 / チームメンバー / 他部門（例：製品部）など】
■報告内容：【商談結果・案件進捗・トラブル発生など（箇条書きでOK）】
■次のアクション：【誰が何をするか】
■共有したい判断・決定事項：【あれば】

件名と本文（300字以内）をセットで作成してください。
```
