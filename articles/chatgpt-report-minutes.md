## 書類仕事は「考える仕事」ではない

営業の仕事で一番時間を使っているのが、提案書でも商談でもなく「書類仕事」という人は多い。

議事録、週次報告書、商談サマリー、社内連絡。これらは「考える仕事」ではなく「整理する仕事」だ。でもなぜか時間がかかる。

ChatGPTはこの「整理する仕事」が得意だ。メモを渡すと、読みやすい形に整えてくれる。

今日は書類仕事に特化したプロンプトを3本紹介する。

---

## プロンプト1｜週次報告書をメモから3分で作る

金曜の夕方、報告書の締め切りが迫っているとき、このプロンプトに今週のメモを貼り付けるだけで完成する。

```
以下のメモをもとに、営業の週次報告書を作成してください。

■今週の訪問先と内容：【会社名・話した内容を箇条書きで】
■今週の数字：【訪問件数、商談件数、受注件数、売上など】
■来週の予定：【主なアポや対応予定を箇条書きで】

箇条書きを中心に、300文字程度でまとめてください。
上司への報告として読みやすい構成にしてください。
```

**メモはどんなに雑でも大丈夫**

「A社 社長と話した。予算は来期。」みたいな走り書きを貼り付けても整理してくれる。

文字数の指定を変えると分量が変わる。上司が細かく読む人なら「500字」、要点だけ欲しい人なら「200字」に変える。

**出力例（メモが走り書きの場合）**

入力メモ：
```
月曜：A社、社長と話した。予算来期。
火曜：B社新規、担当者が親切。提案書送ることになった。
水曜：C社フォロー、まだ検討中。
数字：訪問5社、商談3件、受注なし
来週：B社に提案書送る、D社新規アポ
```

出力：
```
【今週の活動報告】
訪問5社／商談3件／受注0件

＜主な動き＞
・A社（社長）：予算確保は来期予定。継続フォロー中
・B社（新規）：担当者と初回接触。来週中に提案書送付予定
・C社：検討継続中。引き続きフォロー

＜来週の予定＞
・B社への提案書送付
・D社新規アポイント実施
```

## プロンプト2｜商談後の議事録を箇条書きメモから作る

商談中はメモを取るだけで精一杯。帰社後に議事録を整理する時間が惜しい。このプロンプトで、箇条書きのメモをそのまま貼り付けると議事録になる。

```
以下の会議メモをもとに議事録を作成してください。

■会議名：【例：〇〇社との第2回商談】
■日時・参加者：【日時と参加者名（社名・役職）】
■会議メモ：【発言・決定・宿題などを箇条書きで】

以下の形式で出力してください：
- 決定事項（番号付き）
- 次回アクション（担当者・期日つき）
- 確認・検討が必要な事項（あれば）
- 次回会議の予定（わかれば）
```

**「誰が・いつまでに」を毎回明示させる**

議事録で一番重要なのは「誰が何をいつまでにやるか」だ。このプロンプトは常に担当者と期日つきでアクションを出してくれる。

メモに担当者が書いてなければ、ChatGPTは「要確認」と出す。それで商談翌日に担当者を確認する習慣がついた。

## プロンプト3｜社内への報告・連絡メールを2分で作る

商談結果を上司や他部門に共有するメールも、毎回一から書いていると地味に時間を取られる。

```
以下の情報をもとに、社内向け報告メールを作成してください。

■報告相手：【上司 / チームメンバー / 他部門など】
■報告内容：【商談結果・案件進捗・トラブル発生など（箇条書きでOK）】
■次のアクション：【誰が何をするか】
■共有したい判断・決定事項：【あれば】

件名と本文（300字以内）をセットで作成してください。
```

社内メールは外向けメールほど丁寧にしなくていい。でも何を決めたか、次に誰が何をするかは明確に書く必要がある。このプロンプトはそこを自動的に整理してくれる。
