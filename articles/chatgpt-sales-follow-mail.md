## 「送ったのに返事がこない」は営業あるある

提案書を送った。見積書を出した。でも1週間経っても返事がない。

こういうとき、フォローメールを書こうとすると手が止まる。「催促みたいに思われたくない」「でも放置もできない」という板挟みの状態で、結局1時間以上かけて書いたメールが「先日お送りした件いかがでしょうか」という1行になる。

ChatGPTを使えば、この悩みを5分で終わらせられる。

---

## フォローメールで失敗するパターン

よくある失敗は2つ。

**失敗1：催促感が出る**
「ご検討いただけましたでしょうか」という一文は、送った側には無害に見えても、受け取った側には「早く決めろ」に読める。

**失敗2：何を求めているか不明**
「ご確認ください」だけでは、相手は何をすればいいかわからない。返信率が下がる。

プロンプトで「催促にならない」「次のアクションを明示する」を指定すれば、どちらも解決する。

---

## プロンプト1｜未返信先へのフォローアップメール

```
以下の状況でフォローアップメールを作成してください。

■送付内容：【以前送ったもの（例：提案書、見積書）】
■送付日：【日付】
■相手の状況：【わかっている範囲で（例：検討中の様子）】
■目的：【返信を促す / 次の打ち合わせを提案する / 期限確認など】

催促にならない、自然な言い回しで書いてください。
件名と本文をセットで。
```

**使い方のコツ**

「目的」を具体的に書くほど使える文章が出てくる。「返信を促す」より「来週中に可否の返答が欲しい」のほうが、自然にその意図が文章に滲む。

**出力例**

```
件名：先日の提案書について、一点ご確認があります

〇〇株式会社
□□様

先週お送りした提案書、ご確認いただけましたでしょうか。

ご多忙の折、恐れ入りますが、方向性についてご意見を伺えますと幸いです。
もしご不明点があればお気軽にお声がけください。

来週中にお返事いただけますと、スケジュール調整がスムーズに進みますので、
よろしければご一報いただけますと助かります。
```

## プロンプト2｜お断りへの切り返し返信

断られたときのメールほど書きにくいものはない。感情が動くし、何を書いても言い訳や未練に聞こえそうで怖い。

でも実は、断られた後のメールこそ「次の接点」が決まる重要な場面だ。

```
以下の状況でお断りへの返信メールを作成してください。

■断りの内容：【相手のメールの要旨（例：今期は予算確保が難しい旨）】
■自社のスタンス：【今後もアプローチを続けたい / いったん距離を置く】
■トーン：【感謝・共感を前面に、圧力をかけない文体で】

短め（200字以内）で書いてください。件名と本文をセットで。
```

**重要な指定：「短め・感謝を前面に」**

断られた直後に長文を送ると「まだ諦めていない」という印象になる。短く返すことが、実は次につながりやすい。このプロンプトで「200字以内」を指定しているのはそのためだ。

**出力例（予算理由で断られたケース）**

```
件名：ご連絡いただきありがとうございます

□□様

ご丁寧にご連絡いただき、ありがとうございます。
今期はご都合が合わなかったとのこと、承知いたしました。

またご状況が変わりましたら、いつでもお声がけください。
引き続きよろしくお願いいたします。
```

短いほど品がある。

## プロンプト3｜「返信しやすい」メールを作る

フォローメールの返信率を上げる一番の方法は、相手が「はい/いいえ」で答えられる質問を入れることだ。

```
以下の目的で、相手が返信しやすいメールを作成してください。

■目的：【例：来週のアポを取りたい】
■候補日時：【例：来週火曜14時・水曜16時以降】
■相手との関係：【初回商談済み / 既存顧客など】

相手が「はい/いいえ」または選択肢を選ぶだけで返信できる形にしてください。
件名と本文をセットで、150字以内で。
```

「はい/いいえで答えられる質問」を入れるだけで、返信率が体感で1.5倍になる。

---

## まとめ：プロンプトのポイント3つ

1. **「催促にならない」を毎回入れる** → 指定しないと圧力的な文章が出やすい
2. **文字数を指定する** → 短い方が好印象になる場面が多い
3. **相手が選択するだけで返せる構成を要求する** → 返信率が上がる
