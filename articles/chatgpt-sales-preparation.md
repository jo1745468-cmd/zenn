## 商談前の準備に2時間かけていませんか

「明日の商談、どう進めようか」と前日夜に悩む時間。
相手の会社をGoogle検索して、競合を調べて、質問リストを手書きして……。

気づけば2時間経っている。それ、ChatGPTに渡せば10分で終わる。

## なぜ商談準備がうまくいかないのか

準備に失敗するパターンは3つだ。

**パターン1：相手のことを調べすぎて、仮説を立てない**
情報収集に時間をかけたのに、商談では「御社の課題はなんですか？」と聞いてしまう。相手は「それをあなたに教えてほしい」と思う。

**パターン2：自社の説明を準備しすぎる**
自社サービスの説明をA4で3枚用意したが、相手が求めているのは「自分たちの問題が解決するかどうか」だけだ。

**パターン3：質問リストが「確認」になる**
「予算はどのくらいですか」「導入時期はいつですか」は確認であって、会話ではない。相手が「この人と話すと何か気づきがある」と思う質問が一つもない。

ChatGPTはこの3つを正確に指摘して、正しい準備リストを作ってくれる。

## プロンプト1｜ヒアリング仮説を事前に作る

```
以下の情報をもとに、商談前のヒアリング仮説を作成してください。

■商談相手の会社：【会社名・業種・規模（わかる範囲で）】
■商談相手の役職：【例：営業部長】
■商談の目的：【例：SaaSツールの提案】
■自社サービスの概要：【例：営業支援ツール、月額3万円、導入実績200社】

出力形式：
・この会社が抱えていると思われる課題（3つ）
・各課題に対して、自社サービスで解決できること
・商談で確認すべき「仮説の検証ポイント」（各課題ごと）
```

**使い方のポイント**

仮説は「当たる」ために作るのではない。「外れたとき」に相手が話してくれるきっかけにするために作る。

「実は違って、うちの場合は○○なんですよ」という返しが出たとき、商談が本当に始まる。

**出力例（SaaS営業・製造業の中堅企業へのアプローチ）**

```
【課題仮説1】営業担当者ごとの活動量・質にばらつきがある
→ 解決できること：活動ログの可視化と平準化
→ 確認ポイント：現在の営業活動の記録方法、マネージャーの管理負荷

【課題仮説2】商談後のフォローが属人化していて抜け漏れが発生している
→ 解決できること：次アクション自動リマインド、案件管理の一元化
→ 確認ポイント：失注した案件のパターン、商談後の連絡間隔

【課題仮説3】マネージャーが現場の状況をリアルタイムに把握できていない
→ 解決できること：ダッシュボードでのリアルタイム進捗確認
→ 確認ポイント：現在のレポート作成方法、月次会議の内容
```

## プロンプト2｜競合比較表を自動作成する

相手が「他社も見ている」と言われたとき、慌てないために。

```
以下の条件で競合比較表を作成してください。

■自社サービス：【サービス名・特徴・価格帯】
■想定される競合：【社名または「一般的な競合」と記載可】
■顧客が重視する軸（わかれば）：【例：価格・導入速度・サポート体制】

出力：
・比較表（Markdown形式）
・自社の「選ばれやすいポイント」（2〜3点）
・「競合を選ばれやすいシーン」と、それに対する切り返し方
```

**なぜ競合の「負けるシーン」まで作るのか**

相手が他社を選ぶ理由を自分で把握しているかどうかで、商談の深さが変わる。

「価格なら確かにA社の方が安いです。ただ、初期設定に工数がかかるケースが多く、うちの場合は……」という返しができるのは、自分で弱みを把握しているからだ。

## プロンプト3｜相手が「考えさせられる」質問リストを作る

```
以下の商談相手に対して、思考を促す質問リストを作成してください。

■業種・役職：【例：小売業・店長職】
■商談フェーズ：【初回 / 2回目以降 / クロージング前】
■今回確認したいこと：【例：課題の深さ・予算感・決裁プロセス】

条件：
・「はい/いいえ」では答えられない質問
・相手が「考えたことがなかった」と思う切り口を1つ含める
・5〜7問
```

**良い質問と悪い質問の違い**

悪い質問：「現在の課題はなんですか？」
良い質問：「もし今のやり方が来年も変わらなかったとしたら、何が一番困りますか？」

悪い質問は「相手が答えを持っている」前提。良い質問は「答えを一緒に探す」構造になっている。

ChatGPTに「思考を促す質問」と指定すると、後者の方向に引っ張ってくれる。

## プロンプト4｜商談のシナリオを事前に書く

```
以下の条件で商談シナリオを作成してください。

■想定時間：【例：60分】
■商談の目的：【例：課題確認→デモ→次回MTG設定】
■相手の想定スタンス：【例：検討初期、他社も見ている段階】

出力：
・時間割（分刻み）
・各フェーズでの「聞くこと」「話すこと」の優先順位
・商談の最後に必ず確認すること
```

60分の商談は「最初の15分が全て」という話をよく聞くが、それは本当で、最初に相手の関心を掴まなければ残り45分は消化試合になる。

シナリオを書いておくと、どこに力を入れるかが事前に整理できる。
