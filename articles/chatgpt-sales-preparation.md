## 商談前の準備に1時間かけていた

初めて訪問する会社の前日、何時間も準備していた時期がある。

会社のホームページを全ページ読んで、プレスリリースを調べて、業界のことを調べて、聞く質問を考えて。それで当日「思ったより話が弾まなかった」「準備した質問の半分しか使えなかった」という経験を何度もした。

ChatGPTを使い始めてから、商談前の準備は10分になった。かけた時間と商談の質は比例しないと気づいたから、準備の仕方を変えた。

今日は実際に使っているプロンプトを3本紹介する。

## プロンプト1｜業界の基礎を10分で把握する

初めて会う業界のとき、最初にこれを使う。

```
以下の業界・企業について基礎情報を教えてください。

■業界：【例：中小製造業（自動車部品）】
■企業規模：【例：従業員50〜200名程度の中小企業】
■目的：【例：初回商談前の業界理解。課題を仮説立てするため】

以下の形式でまとめてください：
- 業界の主要課題（3点）
- この規模の会社が抱えやすい経営上の悩み（2〜3点）
- 商談で聞くべき確認事項（3〜5点）

※情報が不確かな場合は「一般的に」「傾向として」と断ってください。
```

**注意点**

ChatGPTの情報は学習データの範囲内なので、最新の業界動向は別途確認が必要だ。「仮説を立てるための叩き台」として使う意識が重要。

**使い方**

このプロンプトで出てきた「業界の主要課題3点」を頭に入れて商談に臨む。初回訪問のヒアリングで「御社でも〇〇という課題はありますか？」と聞けるようになる。

準備ゼロで「最近どんな課題をお持ちですか？」と聞くより、仮説を持って確認する質問の方が商談が深まる。

## プロンプト2｜競合他社の情報を5分で整理する

「〇〇社も検討しています」と言われた瞬間、すぐ対応できるように準備しておく。

```
【競合会社名：○○株式会社】について、以下の観点から情報を教えてください。

1. 主力製品・サービスとその特徴
2. 強みとして知られている点
3. 弱みや顧客からの不満として言及されやすい点
4. 価格帯・ターゲット層

※情報が不確かな場合や最新の情報は不正確な可能性があることを明示してください。
回答後、「この情報を踏まえて自社との差別化ポイント」を3点挙げてください
（自社の強みを次に入力します）。
```

このプロンプトを使った後、自社の強みを入力すると差別化ポイントが出てくる。

競合が出てきた商談で「弊社と〇〇社の違いは3点あります」と即答できるようになった。

## プロンプト3｜プレスリリースを商談トークに変換する

訪問先が最近ニュースを出していたとき、それを商談で使えるトークに変換する。

```
以下のニュース/プレスリリースを要約してください。

■テキスト：【記事本文をそのまま貼り付け】

以下の形式で出力してください：
- 要点（3行以内）
- 自社営業への影響・活用できるポイント（あれば）
- 商談で使えるトーク（あれば1〜2文）
```

「商談で使えるトークを出して」という指定が肝だ。これを入れないと、ただの要約になる。

**出力例**

```
要点：
- 〇〇社が新工場を〇月に稼働予定
- 生産能力が1.5倍に拡大
- 品質管理体制の強化も発表

商談で使えるトーク：
「先日の新工場稼働の発表を拝見しました。生産規模が拡大すると、
在庫管理や受発注の負荷も増えると思いますが、その辺りはいかがでしょうか？」
```

プレスリリースを読んだことを示しつつ、自然にヒアリングにつなげられる。

## 準備の「やりすぎ」を防ぐ

ChatGPTで準備が楽になると、逆に準備しすぎるリスクがある。

商談前の準備で本当に必要なのは：
- 業界の主要課題の仮説（3点）
- 競合との差別化ポイント（2〜3点）
- 最初の質問（1〜2個）

これだけで十分だ。

あとは相手の話を聞きながら、その場で考える。準備した情報を全部話そうとすると、ヒアリングができなくなる。
