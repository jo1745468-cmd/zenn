商談が終わって会社に戻ったのが18時。次のアポは翌朝10時。「今日中に提案書送っておきます」と言ってしまった手前、そこから構成を考えて、スライドを開いて、文章を整えて……気づいたら21時を回っていた。そんな夜が、月に何度もありました。

## プロンプト1本目：提案書の構成案を3分で作る

**どんな時に使う？**

商談後、「とりあえず提案書の骨格を今日中に作りたい」というとき。ゼロから考え始めると時間がかかる構成部分を、まず一気に出力させます。

**プロンプト全文**

```
あなたはB2B営業のプロです。以下の情報をもとに提案書の構成案を作成してください。

■提案先：【業種・規模（例：中小製造業・従業員50名程度）】
■提案サービス：【商品名・価格帯（例：クラウド型在庫管理システム・月額3万円〜）】
■課題：【課題内容（例：在庫のダブり発注が月に数回発生している）】
■ページ数：【枚数（例：8枚）】

各セクションのタイトルと、そのページで伝えるべきポイントを箇条書きで出力してください。
```

**ポイント**

「提案先・サービス・課題・ページ数」の4つを渡すだけで、営業的な視点を持った構成案が出てきます。課題欄はできるだけ具体的に書くと出力の質が上がります。

## プロンプト2本目：商談後のお礼メールを5分で仕上げる

**どんな時に使う？**

商談が終わった直後、移動中のスマホや帰社後にすぐお礼メールを送りたいとき。

**プロンプト全文**

```
以下の情報をもとに、商談後のお礼メールを作成してください。

■相手の会社名：【会社名】
■担当者名：【担当者名】
■商談日：【日付】
■商談の内容：【話した内容（箇条書きでOK）】
■トーン：【丁寧・ビジネスライクで。親しみやすさは控えめに】

件名と本文をセットで出力してください。
```

**ポイント**

「トーン」の指定が意外に大事です。相手の業種や関係性に応じて「フォーマル」「一般的なビジネスメール」など変えると、ほぼそのまま使える文章になります。

## プロンプト3本目：週次報告書を箇条書きメモから完成させる

**どんな時に使う？**

毎週金曜日の報告書提出。手元のメモから素早くまとめたいとき。

**プロンプト全文**

```
以下のメモをもとに、営業の週次報告書を作成してください。

■今週の訪問先と内容：【会社名・話した内容を箇条書きで】
■今週の数字：【訪問件数、商談件数、受注件数、売上など】
■来週の予定：【主なアポや対応予定を箇条書きで】

箇条書きを中心に、300文字程度でまとめてください。
上司への報告として読みやすい構成にしてください。
```

**ポイント**

文字数指定を変えると分量が変わります。「500字」にすると詳細版、「200字」にすると要点のみの版になります。
