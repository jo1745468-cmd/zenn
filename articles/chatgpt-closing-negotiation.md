## クロージングで「うまく言えなかった」は準備不足です

商談は良かった。でもクロージングでうまく言葉が出てこなかった。値引き交渉をされたとき、慌てて「少し検討します」と言ってしまった。

こういう失敗は、事前に「言葉を用意していなかった」だけのことが多い。

ChatGPTで想定シナリオを作っておけば、クロージングは「言葉が出てこない」問題ではなくなる。

---

## クロージングで詰まる3つの場面

**場面1：「もう少し検討します」と言われたとき**
検討期間が長くなれば決裁者が変わる可能性がある。でも「早く決めてください」とは言えない。

**場面2：「もう少し安くなりませんか」と言われたとき**
安易に値引きすると信頼を失う。断ると関係が壊れる気がする。

**場面3：「他社と比べています」と言われたとき**
「うちの方がいいです」と言っても響かない。何を言えばいいのかわからない。

この3つのシナリオを事前にChatGPTで練習しておく。

---

## プロンプト1｜「検討します」への対応トーク

```
以下の状況で「もう少し検討します」と言われました。
自然で圧力をかけない形で、次のステップに進むためのトークを作成してください。

■商談の状況：【例：2回目商談、相手は前向きだが決裁者への確認が必要と言っている】
■自社サービス：【概要・価格帯】
■相手の懸念として想定されること：【例：予算・上長の承認・タイミング】

条件：
・「いつ決まりますか」という催促にならないこと
・次回の接点を自然に作ること
・返答のパターンを2〜3案
```

**出力例**

```
【案1】確認事項を絞る
「ありがとうございます。もし上長へのご説明の際に資料が必要でしたら、
ご担当者様向けと決裁者様向けの2バージョン用意できます。
どちらが使いやすそうですか？」

【案2】懸念点を引き出す
「検討いただけるとのことで、ありがとうございます。
率直にお聞きしてもよいですか。今の段階で気になっている点が
あれば教えていただけると、より的確な情報をお伝えできます」

【案3】次回を具体的に設定する
「わかりました。来週あたりに少しだけお時間いただいて、
ご不明な点があれば解消できればと思っています。
火曜か水曜はいかがでしょうか」
```

## プロンプト2｜価格交渉への切り返しトーク

```
以下の状況で「もう少し安くなりませんか」と言われました。
値引きせずに納得していただくためのトークを作成してください。

■提示価格：【例：月額10万円】
■サービスの強み：【例：導入後の専任サポート、設定代行、解約率2%】
■相手が値引きを求める背景（推測）：【例：予算が厳しい / 競合と比較している】

条件：
・値引きを即答しないこと
・価格の根拠を感情的でなく事実で説明すること
・相手の懸念に共感してから話すこと
```

**値引き交渉で大事なこと**

値引きに応じるかどうかよりも先に、「なぜ安くしてほしいのか」を理解することが重要だ。

予算が本当に足りないのか、競合と比べているのか、値引き交渉が文化的な習慣なのかによって、対応が変わる。

ChatGPTに「背景ごとに分けて」と指定すると、場合分けしたトークを作ってくれる。

**出力例**

```
【予算が厳しい場合】
「おっしゃる通り、初期投資として大きな判断ですよね。
一つご提案なのですが、まず3ヶ月の試験導入という形で始めて、
効果を確認してから継続という進め方はいかがでしょうか。
その場合、初期コストを抑えた形でご提案できます」

【競合比較の場合】
「比較いただいているとのこと、ありがとうございます。
もし差額に見合うかどうかをご判断いただくために、
導入後の工数削減の試算を出させてください。
半年間で見ると、実質コストが逆転するケースが多いです」
```

## プロンプト3｜競合比較をされているときのトーク

```
以下の状況で「他社さんとも話を進めています」と言われました。
競合を否定せずに自社を選んでもらうためのトークを作成してください。

■自社の強み（客観的事実）：【例：導入社数・サポート体制・特定業界への実績】
■競合の一般的な特徴：【例：価格が安い、機能がシンプル】
■この顧客が最終的に重視しそうなこと：【例：導入後の定着率・コストより効果】

条件：
・競合を直接批判しない
・「うちを選ぶと何が変わるか」を具体的に示す
・相手が自分で判断できる情報を提供する形にする
```

**「比較してください」というスタンスが強い**

「うちが一番です」と言う営業と「しっかり比較して決めてください、その上でこれだけお伝えしたいことがあります」と言う営業では、後者の方が信頼される。

プロンプトに「自分で判断できる情報を提供する形」と入れるのはそのためだ。

## プロンプト4｜クロージング練習（ロールプレイ）

準備のもう一つの方法は「練習」だ。

```
私は営業担当で、あなたは「検討します」と言いそうな顧客役をしてください。

■顧客の設定：【業種・役職・懸念ポイント】
■私のサービス：【概要】

まず顧客側からの発言を1つ書いてください。
私が返答したら、顧客として反応してください。これを3往復繰り返してください。
```

商談本番で初めて「検討します」に対応するのと、3回練習してから臨むのでは全然違う。

ChatGPTはロールプレイのパートナーとして優秀で、顧客の反応も自然だ。

## 準備があれば、クロージングは怖くない

クロージングで詰まるのは、技術の問題ではなく準備の問題であることが多い。

「何を言うか」があらかじめ頭に入っていれば、商談中は「聞く」ことに集中できる。
