## 新規開拓の「最初の15分」に全力を使っていますか

「この会社に電話する前に少し調べよう」と思い、結局1時間かけても「で、何を話せばいいんだっけ」という状態になることがある。

リサーチに時間をかけることと、リサーチが深まることは別の話だ。

ChatGPTを使えば、業界の傾向・想定課題・最初のアプローチ文まで、15分で揃う。

---

## 新規開拓で時間が消える3つの罠

**罠1：会社のホームページを読み続ける**
ホームページからわかることは「その会社が見せたいこと」だ。課題はそこには書いていない。

**罠2：業界全体を調べようとする**
「小売業全体の動向は……」から入ると、個社への仮説に辿り着くまでに時間がかかる。

**罠3：最初のメールをゼロから書こうとする**
初回アプローチメールは、フォーマットが決まっている。ゼロから書く必要はない。

---

## プロンプト1｜業界×役職の課題仮説を作る

```
以下の条件で、初回アプローチのための課題仮説を作成してください。

■ターゲット業界：【例：製造業（中小企業、従業員100〜500名）】
■アプローチ先の役職：【例：総務部長・経営企画担当】
■自社サービス・商品：【概要・価格帯・解決できること】

出力：
・この業界・役職が抱えやすい課題（上位3つ）
・各課題が発生する背景（なぜそうなっているか）
・自社サービスで解決できるポイント
・初回アプローチで使える「課題提起フレーズ」（各課題につき1つ）
```

**出力例（製造業・総務部長へのSaaS営業）**

```
【課題1】ペーパーレス化が進まず、書類管理の工数が多い
背景：現場のITリテラシーにばらつきがあり、デジタル化が定着しにくい
解決ポイント：操作が簡単なUIと導入時の研修サポート
アプローチフレーズ：
「製造業様では書類のデジタル化に取り組まれているケースが増えていますが、
現場への定着でつまずくことが多いとお聞きしています。御社でも同様の
課題がありますか？」

【課題2】複数拠点での情報共有に時間がかかる
背景：拠点ごとに管理システムが異なる、メール・電話が主な手段
解決ポイント：クラウドによるリアルタイム共有機能
アプローチフレーズ：
「複数拠点をお持ちの場合、情報のタイムラグが品質管理に影響することが
多いとお聞きしています。御社ではどのように情報共有されていますか？」
```

## プロンプト2｜初回アプローチメールを作る

課題仮説が作れたら、次は最初のメールだ。

```
以下の条件で初回アプローチメールを作成してください。

■ターゲット：【会社名・担当者役職（名前不明でもOK）】
■課題仮説：【プロンプト1で作ったものをコピー】
■自社の提案内容：【サービス概要・具体的な提案】
■目的：【例：資料送付の許可 / 15分の電話打ち合わせ依頼】

条件：
・件名は相手がメリットを感じられる内容にする
・本文は300字以内
・次のアクションを1つだけ提示する（複数提示しない）
```

**300字以内にする理由**

初回メールを読む相手は、あなたのことを知らない。知らない人から来た長いメールを読む理由がない。

「何者かわかること」「相手に何があるかわかること」「次にすることが1つだけあること」の3つが揃えば、返信率が上がる。

**出力例**

```
件名：製造業の書類管理工数を40%削減した事例をご紹介できます

○○株式会社
総務ご担当者様

突然のご連絡失礼いたします。株式会社△△の□□と申します。

製造業のお客様から「書類管理に現場の工数がかかりすぎる」という
課題をよくお聞きしており、同業他社での削減事例をまとめた資料がございます。

5分程度でご確認いただける内容ですので、ご興味があればお送りできます。
ご返信いただけますでしょうか。
```

## プロンプト3｜初回電話のスクリプトを作る

メールより電話の方が接触率は高い。ただ「最初の10秒」を失敗すると切られる。

```
以下の条件で、新規開拓の電話スクリプト（最初の30秒）を作成してください。

■自社・氏名：【例：株式会社△△の□□】
■ターゲット：【業種・役職】
■課題提起：【相手に刺さりやすい課題をひとつ】
■目的：【例：メール送付の許可を得る / 5分だけ話を聞いてもらう】

条件：
・最初の10秒で「なぜかけてきたか」を明確にする
・断られた場合の切り返しを2パターン含める
```

**電話で大事なこと：目的を絞る**

初回電話で「商品説明をしよう」とすると失敗する。

初回電話の目的は「次の接点を作ること」だけだ。「資料を送らせてください」「5分だけお時間ください」が取れれば成功。この意識があるだけで、電話の成功率が変わる。

## プロンプト4｜競合他社のいない切り口を探す

同じ業界に同じアプローチをしている競合がいる場合、差別化が必要だ。

```
以下の条件で、競合との差別化ポイントを整理してください。

■自社サービス：【概要・強み・弱み】
■想定される競合：【2〜3社または「一般的な同業他社」】
■ターゲット顧客層：【業種・規模・役職】

出力：
・競合と被らない切り口（アプローチの角度）
・この切り口が刺さりやすい顧客の特徴
・使えるキャッチコピー（2〜3案）
```

**「特定の業種特化」が最も使いやすい差別化**

「製造業に特化した営業支援」「飲食店に特化した予約管理」のように、業種を絞るだけで競合が少なくなることが多い。

総合的に強い競合に対して、特定業種の深さで勝つ。これが中小企業の営業戦略としてよく機能する。
