## 提案書作りで「何から書けばいいかわからない」が続いていませんか

提案書を作るとき、一番時間を取られるのは「書く」作業ではなく「構成を考える」作業だ。

何を先に書くか。どのデータを使うか。相手の課題をどう言語化するか。これを考えているだけで2時間消える。

ChatGPTはこの「構成を考える」部分が得意だ。方向性だけ渡せば、提案書の骨格を1分で返してくる。

---

## 提案書に必要な要素は5つだけ

長くなりがちな提案書だが、構成として必要な要素は5つに絞られる。

1. **現状の課題**（相手の言葉で、数字があれば理想）
2. **放置した場合のリスク**（なぜ今解決すべきか）
3. **解決策と根拠**（自社サービスの内容と実績）
4. **導入後の変化**（具体的に何が変わるか）
5. **投資対効果とアクション**（費用と次のステップ）

この5つを埋めれば提案書になる。ChatGPTはこの5つを「相手に合わせた言葉」で埋めてくれる。

## プロンプト1｜提案書の構成と見出しを作る

```
以下の情報をもとに提案書の構成と各スライドの見出し・要点を作成してください。

■提案先：【会社名・業種・担当者役職】
■相手の課題（ヒアリングで聞いた内容）：
【例：営業担当者ごとの受注率にばらつきがある、トップセールスのノウハウが共有できていない】

■自社サービス：【概要・価格・導入実績】
■今回の提案の目的：【例：試験導入の提案、本契約の提案】

出力：
・スライド構成（1枚ごとに見出し＋要点2〜3行）
・提案書全体のストーリーライン（なぜこの順番か）
```

**ストーリーラインが大事な理由**

提案書は「情報の羅列」ではなく「納得のストーリー」でなければならない。

「課題がある→放置するとこうなる→解決策がある→効果はこれだ→今すぐ動こう」という流れが、相手の頭の中で完成したとき、承認が出る。

ChatGPTに「なぜこの順番か」まで説明させると、ストーリーとして成立しているかを確認できる。

## プロンプト2｜課題スライドの文章を作る

```
以下の情報をもとに、提案書の「現状の課題」スライドの文章を作成してください。

■ヒアリングで把握した課題：
【例：月間の商談件数は平均20件だが、受注率は担当者によって10〜35%とばらつきがある】

■数字や事実（わかれば）：
【例：トップセールスの受注率35%、平均受注率18%】

■相手が課題として認識しているか：
【認識している / なんとなく感じている / まだ気づいていない】

出力：
・スライドのタイトル
・課題を表現するメインコピー（20字以内）
・説明文（3〜4行）
・データの使い方の提案（グラフや図にするなら何が効果的か）
```

**「相手の言葉で書く」ことが最重要**

提案書の課題スライドで失敗するのは、「自社視点」で書いてしまうパターンだ。

「弊社のサービスを使っていない企業様は〜」ではなく、「ヒアリングでおっしゃっていた通り、受注率のばらつきが〜」と書く。

相手の言葉がそのまま出てくると、「わかってもらえている」感が生まれる。ChatGPTにヒアリング内容をそのまま貼ると、この感覚を出してくれる。

## プロンプト3｜投資対効果（ROI）の計算と文章化

数字で語れる提案書は強い。

```
以下の情報をもとに、投資対効果を計算し、提案書に使える形式で表現してください。

■導入コスト：【例：初期費用30万円 + 月額10万円】
■期待できる効果（ヒアリングまたは他社事例ベース）：
【例：営業担当者10名の報告書作成時間が週5時間→2時間に短縮】

■担当者の平均時給（概算）：【例：3,000円/時間】
■効果が出るまでの期間：【例：導入後3ヶ月】

出力：
・月次コスト削減額の計算
・年間換算での投資回収シミュレーション
・提案書に使えるROI説明文（2〜3行）
```

**出力例**

```
【ROI計算】
削減工数：10名 × 3時間/週 × 4週 = 月120時間
削減コスト換算：120時間 × 3,000円 = 月36万円
月額コスト：10万円
月次純効果：+26万円

年間換算：+312万円（初期費用30万円は1.1ヶ月で回収）

【提案書文章】
現在、営業担当者10名の報告書作成に費やしている工数は月120時間。
本サービスの導入により、この工数を3分の1に削減できます。
月額コスト10万円に対し、月間効果は36万円。導入から約5週間で投資を回収できます。
```

## プロンプト4｜反論への回答を先に準備する

提案書を出した後に来る質問を、先に準備しておく。

```
以下の提案に対して、相手から出やすい反論・質問を予測し、
それぞれへの回答を準備してください。

■提案内容：【概要】
■提案先：【業種・規模・役職】
■価格：【初期費用・月額】

出力：
・想定される質問・反論（5〜7個）
・各質問への回答（事実ベース、感情的にならない表現で）
```

**一番多い反論：「今は時期じゃない」**

「今は時期ではない」という反論は、「課題は認めるが決断したくない」というサインであることが多い。

この反論への回答は「なぜ今なのか」を数字で示すことだ。「半期のタイミングで始めると、来期の評価に間に合います」「今月始めると、12月の繁忙期前に定着できます」という形で、タイミングの理由を具体化する。

## プロンプト5｜提案書の表紙・エグゼクティブサマリーを作る

決裁者が最初に見るのは表紙とサマリーだ。

```
以下の情報をもとに、提案書の表紙と1ページ目のエグゼクティブサマリーを作成してください。

■提案先：【会社名】
■提案タイトル案：【例：営業生産性向上のための〇〇導入提案】
■提案の一言まとめ（どんな課題をどう解決するか）：
■期待効果（数字で）：
■次のアクション：

出力：
・表紙のキャッチコピー（25字以内）
・エグゼクティブサマリー（決裁者が3分で読める150字以内）
```

決裁者は提案書を最初から最後まで読まない。表紙とサマリーと最終ページだけ見ることが多い。

逆に言えば、この3ページが完成していれば、承認は取れる。

## まとめ：提案書を1時間で仕上げる流れ

| ステップ | プロンプト | 時間 |
|---|---|---|
| 1. 構成・骨格を作る | プロンプト1 | 10分 |
| 2. 課題スライドを作る | プロンプト2 | 10分 |
| 3. ROIを計算・文章化 | プロンプト3 | 10分 |
| 4. 反論への回答を準備 | プロンプト4 | 10分 |
| 5. 表紙・サマリーを作る | プロンプト5 | 10分 |
| 6. 微調整・整形 | 手作業 | 10分 |

合計1時間。提案書を作る本当の目的は「承認を得ること」だ。そのために必要な時間を最小化する。
